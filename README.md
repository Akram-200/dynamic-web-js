<h1>Projet JavaScript – Formulaire sécurisé (Validation + Responsive)</h1>

<h2>Description</h2>
<p>
Projet front-end visant à démontrer :
</p>
<ul>
  <li>Validation côté client en <strong>JavaScript</strong></li>
  <li>UX simple (messages d’erreur clairs, compteur de caractères)</li>
  <li>Interface <strong>Responsive Design</strong> (mobile et desktop)</li>
  <li>Petite protection côté client (sanitization) et bonnes pratiques</li>
</ul>

<h2>Fonctionnalités</h2>

<h3>Validation</h3>
<ul>
  <li>Nom complet (minimum 3 caractères)</li>
  <li>Email (format valide)</li>
  <li>Téléphone optionnel (10 chiffres)</li>
  <li>Sujet obligatoire</li>
  <li>Message (entre 10 et 300 caractères)</li>
</ul>

<h3>Interface utilisateur (UI)</h3>
<ul>
  <li>Erreurs affichées sous chaque champ</li>
  <li>Compteur de caractères (0 / 300)</li>
  <li>Bouton de réinitialisation</li>
  <li>Aperçu des données (simulation d’envoi)</li>
</ul>

<h2>Installation / Lancement</h2>
<ul>
  <li>Télécharger ou cloner le dépôt</li>
  <li>Ouvrir le fichier <code>index.html</code> dans un navigateur</li>
</ul>
